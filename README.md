# React Weather App

A simple Weather App built using React.js and OpenWeather API.

## Features

- Search weather by city name
- Real-time weather information
- Temperature display in Celsius
- Weather condition and description
- Dynamic weather icons
- Error handling for invalid cities
- Responsive user interface

## Technologies Used

- React.js
- JavaScript (ES6+)
- CSS3
- OpenWeather API

## Concepts Learned

- React Components
- useState Hook
- API Fetching
- Async/Await
- Conditional Rendering
- Event Handling
- Error Handling

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
npm install
```

3. Start development server

```bash
npm run dev
```

## Project Goal

The main goal of this project was to learn how React applications communicate with external APIs and display dynamic data to users.
